Provides various short bash scripts for use with University of Waterloo CS Servers.  

---

I recommend adding these scripts to PATH:  
Linux: https://unix.stackexchange.com/questions/26047/how-to-correctly-add-a-path-to-path  
Windows: https://stackoverflow.com/questions/44272416/add-a-folder-to-the-path-environment-variable-in-windows-10-with-screenshots  
Mac: https://stackoverflow.com/questions/22465332/setting-path-environment-variable-in-macos-permanently

# valgrind_ssh.sh
Submit files to Valgrind via SSH  
Usage: `./valgrind_ssh.sh <file>`

# marmoset_submit.sh
Submit files to Marmoset via SSH  
Usage: `./marmoset_submit.sh <course> <project> <file1> [file2] [file3] ...`
